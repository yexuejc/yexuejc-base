通用工具包

### 使用
>yexuejc.base.version=1.0.0

pom.xml
```
<dependencies>
    <dependency>
        <groupId>com.github.yexuejc</groupId>
        <artifactId>yexuejc-base</artifactId>
        <version>${yexuejc.base.version}</version>
    </dependency>
</dependencies>
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
